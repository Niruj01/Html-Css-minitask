10) In your, HTML page add the below line and Highlight it without using any CSS.
    "HTML & CSS is awesome"

<!DOCTYPE html>
<html lang="en">

    <meta charset="UTF-8">

    <title>Highlight text</title>
</head>

</body>
</html>
